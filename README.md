# Google Classroom API

The Google Classroom API provides programmatic access to Google Classroom, enabling management of courses, coursework, student submissions, rosters, and invitations. It supports creating and organizing courses, distributing assignments and materials, managing student and teacher enrollments, tracking submissions and grades, and integrating with third-party educational tools through add-on attachments.

## Base URL

```
https://classroom.googleapis.com
```

## Resources

The Google Classroom API v1 provides the following resources:

- **Courses** - Create, read, update, and delete courses
- **Course Work** - Manage assignments and graded work
- **Student Submissions** - Track and grade student work submissions
- **Students** - Manage student enrollment in courses
- **Teachers** - Manage teacher assignments to courses
- **Announcements** - Post and manage class announcements
- **Course Work Materials** - Distribute instructional resources
- **Topics** - Organize course content by topic
- **Rubrics** - Define grading rubrics for course work
- **Invitations** - Send and manage course invitations
- **User Profiles** - Access user account information
- **Guardians** - Manage parent/guardian associations
- **Registrations** - Configure push notifications via Cloud Pub/Sub
